# Golsa Ghasemi academic website

I keep this repository as the source for Dr. Golsa Ghasemi's academic website at [golsagh94.github.io](https://golsagh94.github.io).

The site is a small Jekyll project for GitHub Pages. It presents her medical education, clinical and research experience, publications, presentations, guideline work, and professional contact links.

## Run locally

Install the Ruby gems:

```bash
bundle install
```

Start a local preview:

```bash
bundle exec jekyll serve
```

Build the site:

```bash
bundle exec jekyll build
```

GitHub Pages serves this user site from the root of the `main` branch.
